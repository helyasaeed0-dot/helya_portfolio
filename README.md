# 🌐 Helya Portfolio

A modern, responsive, and interactive developer portfolio showcasing my skills, projects, and experience as a **Full Stack MERN Developer**. Built with performance, clean design, and user experience in mind, this portfolio highlights my work and provides an easy way for recruiters and clients to connect with me.


---

# 📖 Overview

Helya Portfolio is a personal portfolio website designed to showcase my technical skills, featured projects, certifications, and professional journey. The website follows modern UI/UX principles, ensuring a fast, responsive, and engaging experience across all devices.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/helya-portfolio.git
```

Navigate to the project

```bash
cd helya-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
